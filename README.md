# Nav4Agents

AI Agent 工具导航门户，发现、收藏优质 MCP 服务器和 AI Skills。

## 功能特性

- 🔍 MCP 服务器浏览与搜索
- 🧩 AI Skills 发现
- 📱 响应式设计，支持移动端
- ⚡ 基于 Next.js 14 + Tailwind CSS

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 启动生产服务
npm start
```

## 技术栈

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- TypeScript
